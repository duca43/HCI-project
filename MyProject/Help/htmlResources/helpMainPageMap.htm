World Map

In the center of the main window is placed World Map. Map is used to display monuments. Monuments on map are represented just with their icon. More information about monument can be given by putting mouse over icon. Then the text with information pops up as a tooltip. It gives information about monument's ID (Sequence of characters which is unique within all monuments), name and type.

Monuments can be put on the map by using drag and drop method. They can be dragged from List view of monuments. Dragged monument can be dropped anywhere on the map. Only constraint about dropping monument is that two monuments can't be dropped on the same position. Dropped monument can be moved on the other position by dragging its icon and dropping it on desired new position. User can also zoom the map to see some monument or some country position better. Zooming of the map can be done by pulling the zoom slider. Zoom slider is placed in Application's footer. Drag and drop example is displayed below.
