Add or edit monument type

Monument type is described with:

  1. ID - Sequence of characters which is unique within all monument types. Maximum 16 characters.
  2. Name - Represents not unique name of the monument type. Maximum 50 characters.
  3. Description - A brief description of the monument type. Maximum 250 characters.
  4. Icon - The icon used to visualize a monument type. Image in PNG, JPG or JPEG format, choosed from user file system.

All fields are required except for the description. Icon is required because monuments without icon use icon of their type. ID, name and description can contain any character supported by Unicode standard (UTF-16). On the right side of description text field there is an Icon Preview field. When the icon is choosed, it is displayed in icon preview field.
