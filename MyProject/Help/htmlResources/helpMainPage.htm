About application

World Nautral Monuments is an application for keeping records of the geographical distribution of places of outstanding natural beauty. Application supports manipulating with 3 entities: Monument, Monument type and Tag. All entities are organized in collections. It is possible to manipulate with these collections with 3 primary operations: add, edit and delete entity. Every monument belongs to some monument type. Monuments can also be tagged with no one, one or more tags. Tag can be given to no one, one or more monuments. More about choosing tags read here.
