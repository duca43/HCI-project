List of Monuments

On the left side of the main window is placed List of monuments. List displays all existing monuments in application. Monuments are represented by their icon and ID (Sequence of characters which is unique within all monuments). More information about them gives the tooltip which pops up on putting the mouse over some monument in the list. It gives information about monument's ID, name and type.

List view of monuments is used not only for representing monuments but also for dragging monuments in drag and drop process. More about drag and drop of monuments find on page about World Map. Also, user can quickly access edit dialog of monument by selecting one of them and pressing keyboard shortcut Ctrl+E or by left double-click on it. Deleting monument can be done by pressing Delete button on keyboard. Other way that edit or delete can be performed is by opening context menu with the right click.
