ADD MONUMENT TYPE DIALOG

This dialog you have to use to enter data about new monument type. You have to enter data in order. At first, enter ID (Sequence of characters which is unique within all monument types maximum 16 characters). After that, enter Name, then Description and in the end you have to choose Icon for new monument type. Click on the Browse button and then choose image (PNG or JPG format) from your file system. After you have chosen image, click on Add button to add new monument type in your data collection.
