Display of the monuments

All monuments in application are displayed in the table. Each row represents one monument. There are two modes of displaying data - collapsed or expanded. Mode can be switched with click on the button in the lower left corner (See on the two images bellow the text). In collapsed mode is displayed just the most important data: Icon, ID, Name and Type. In expanded mode is displayed all of the data about monuments.

Next to the Expand/Collapse button are there more buttons: Add, Edit and Delete. Add button opens a dialog for adding a new monument. Edit button opens a dialog for editing a monument which is selected in table. Opening a monument edit dialog can also be performed by left double-click on it in the table. Delete button performs delete of the monument which is selected in table. It can also be performed by pressing Delete button on the keyboard. If there are selected more than one monument, all of them will be deleted. Before deleting a monument user is asked is he sure he wants to delete the monument(s). Other way that edit or delete can be performed is by opening context menu with the right click on the monument in the table.

In the upper right corner there is a list of tags to which the currently selected monument is tagged. Bellow the list is a text field Search. It can be used to search for the one or more monuments by their ID. Bellow the search text field is placed Filter data section. Data displayed in the table can be filtered by 4 different parameters: type, tags, climates and tourist status. For each parameter it can be selected none, one or more values. If there is no selected value for some parameter, monuments with all values of that parameter will be displayed by default. After selecting by which values to filter the data, click on the Filter button. Selected values can be reset by clicking on the Reset button.
