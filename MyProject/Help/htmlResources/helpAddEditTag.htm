Add or edit tag

Tag is described with:

  1. ID - Sequence of characters which is unique within all tags. Maximum 16 characters.
  2. Description - A brief description of the tag. Maximum 250 characters.
  3. Color - The color used to visualize a tag.

The only required field is ID. ID and Description can contain any character supported by Unicode standard (UTF-16). In a dialog for adding a new tag, color is set to default value - white. User can pick any kind of color by using a color picker Tags can be used for tagging one or more monuments.
