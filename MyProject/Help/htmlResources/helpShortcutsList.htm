List of keyboard shortcuts

Keyboard shortcut Descrption
F1 Access help section.
F2 Turn on tutorial mode.
Alt+F4 Exit application.
Ctrl+M Open dialog for creating
new monument.
Ctrl+Shift+M Open dialog for creating
new monument type.
Ctrl+T Open dialog for creating
new tag.
Ctrl+E Open dialog for editing
*selected monument.
Delete Delete *selected monument.
Ctrl+R Remove all monuments from the map.
Ctrl+1 Open window with table that
diplays all monument types.
Ctrl+2 Open window with table that
diplays all monuments.
Ctrl+3 Open window with table that
diplays all tags available.

*Selected monument means that monument has to be selected in list view on the left side of the main window.
