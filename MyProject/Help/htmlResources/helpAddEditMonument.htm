Add or edit monument

Monument is described with:

  1. ID - Sequence of characters which is unique within all monuments. Maximum 16 characters.
  2. Name - Represents not unique name of the monument. Maximum 50 characters.
  3. Description - A brief description of the monument. Maximum 250 characters.
  4. Type - Monument type, one of the three entities that user can manipulate with.
  5. Climate - The climate in which monument is located.
  6. Icon - The icon used to visualize a monument. Image in PNG, JPG or JPEG format, choosed from user file system.
  7. Ecologically Endangered - Determines whether the monument is ecologically endangered.
  8. Habitat - Determines whether the monument is habitat of endangered species.
  9. Populated region - Determines whether the monument is located in populated region.
  10. Tourist Status - Determines the tourist availability of the monument.
  11. Income - Represents the total revenue from the monument.It is a real number.
  12. Discovered Date - The date when the monument was discovered.

It is not necessary to enter all parameters mentioned above to add a new monument in application. Required fields to fill are: ID, Name and Income. ID, name and description can contain any character supported by Unicode standard (UTF-16). Income is a real number and currency of income is a dollar. Climate and tourist status can be choosed by using a combo box and type by using the editable combo box. On the right side of description text field, there is button Choose tags.. Clicking on that button opens a dialog for tagging a monument. Monument can be tagged with no one, one or more tags. Bellow the button for choosing tags there is an Icon Preview field. When the icon is choosed, it is displayed in the icon preview field. If icon is not set, it will be used the icon of the monument's type.
