Display of the monument types

All monument types in application are displayed in the table. Each row represents one monument type. In the table are displayed all parameters of the monument type in this order: Icon, ID, Name and Description. There are there buttons in the lower left corner: Add, Edit and Delete. Add button opens a dialog for adding a new monument type. Edit button opens a dialog for editing a monument type which is selected in table. Opening a monument type edit dialog can also be performed by left double-click on it in the table. Delete button performs delete of the monument type which is selected in table. It can also be performed by pressing Delete button on the keyboard. If there are selected more than one monument type, all of them will be deleted. Before deleting a monument type user is asked is he sure he wants to delete the monument type(s). Other way that edit or delete can be performed is by opening context menu with the right click on the monument type in the table.
