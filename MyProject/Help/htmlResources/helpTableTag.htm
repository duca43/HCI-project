Display of the tags

All tags in application are displayed in the table. Each row represents one tag. In the table are displayed all parameters of the tag in this order: Color, ID, and Description. There are there buttons in the lower left corner: Add, Edit and Delete. Add button opens a dialog for adding a new tag. Edit button opens a dialog for editing a tag which is selected in table. Opening a tag edit dialog can also be performed by left double-click on it in the table. Delete button performs delete of the tag which is selected in table. It can also be performed by pressing Delete button on the keyboard. If there are selected more than one tag, all of them will be deleted. Before deleting a tag user is asked is he sure he wants to delete the tag(s). Other way that edit or delete can be performed is by opening context menu with the right click on the tag in the table.
